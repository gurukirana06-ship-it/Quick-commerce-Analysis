#Project Overview

This project is an end-to-end Data Analytics and Business Intelligence project focused on analyzing Quick Commerce sales, customer purchasing behavior, delivery performance, and operational efficiency.

The project involved data cleaning, preprocessing, and transformation in Microsoft Excel, followed by exploratory data analysis using Pivot Tables and Pivot Charts. Power BI was then used to build an interactive dashboard that provides comprehensive insights into sales performance, customer trends, product demand, delivery efficiency, and city-wise operations.

Through this project, various analytics techniques such as KPI analysis, trend analysis, customer segmentation, category performance evaluation, delivery performance monitoring, and business intelligence reporting were applied to identify growth opportunities and operational improvements.
